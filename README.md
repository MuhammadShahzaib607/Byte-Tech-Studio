# ByteTechStudio

A modern digital agency website designed to bridge the gap between clients and high-quality digital services, featuring an integrated AI-powered business assistant.

## Description

ByteTechStudio is a startup founded by two young developers from Pakistan with a vision to lead in the AI era. The platform serves as a professional hub where clients can explore our services and connect with us. It features a custom-trained AI Chatbot that acts as a 24/7 consultant, providing instant information about our business, services, and goals to any visitor.

Check out the live project here: [https://byte-tech-studio.vercel.app/](https://byte-tech-studio.vercel.app/)

## Tech Stack

* **Frontend:** Next.js, Tailwind CSS, Framer Motion, Axios
* **Backend:** Node.js, Express, UUID, Groq API
* **Database:** MongoDB with Mongoose
* **Deployment:** Vercel, GitHub

## Core Features

* **AI Business Assistant:** A dedicated chatbot trained via the Groq API that knows everything about ByteTechStudio and can answer any user query regarding our services.
* **Service Showcase:** A dedicated page highlighting our core offerings, including MERN stack development and AI-powered applications.
* **Founder Insights:** An About page sharing our story as young Pakistani entrepreneurs and our mission to innovate within the AI space.
* **Modern UI:** A fully responsive and animated interface built for a professional user experience.

## Setup Instructions

To run the project locally for testing or development, follow these steps:

1. Clone the repository

```bash
git clone https://github.com/MuhammadShahzaib607/Byte-Tech-Studio.git
```

2. Install dependencies

```bash
npm install
```

3. Start the application

```bash
npm run dev
```
